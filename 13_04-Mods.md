- No public RSS feed at any conventional path
- Content hub (`zero100.com/content-hub/`) requires membership login
- JS-based bot verification blocks all scraping attempts
- Content is primarily podcast episodes + gated intelligence reports

**Action**: Add a formal entry in `config/sources.yaml` under Tier 2 `analyst` category (alongside Gartner, Hackett Group) as `active: false`, modeled after the `Procurement Leaders` pattern (paid membership — may never be scrapable). Also update the discovery gap comment block to note zero100 has been evaluated.

**File to modify**: `config/sources.yaml`

Add after existing analyst entries (around line 120–140, near Gartner/Hackett):
```yaml

    url: "https://zero100.com/content-hub/"
```
